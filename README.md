# CountriesApp

It is an application that contains many up-to-date android jetpack libraries and can display country details both online and offline.

   
### Architecture
- MVVM Design Pattern 
    
   
## Used in the project
- Firebase
- Coroutines
- Retrofit
- RxJava
- Room
- Data Binding
- Advanced SharedPreferences
- Extension
